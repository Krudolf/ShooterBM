Para ejecutar la prueba en un servidor dedicado se debe primero configurar los ficheros *.bat con las rutas absolutas a los ejecutables del editor de Unreal Engine (version 4.25) y al del proyecto.

De las 6 secciones que componen la prueba, se han completado todas menos la sección 4, la cual trata de crear un widget de vida para el jugador.
El widget funciona de forma correcta en el propio cliente, pero no se replica el estado a los otros clientes (esto se puede probar mirando hacia arriba para ver el widget y con otro cliente dispararle a este primero).

Para probar la habilidad de la sección 5, el jugador debe de mantener pulsada la tecla 'Q' del teclado.
